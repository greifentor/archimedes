Hit Hilfe dieses Packages wird eine effektive Verwaltung von Properties und die Generierung einer Klasse realisiert, die die verwalteten Properties zu Bearbeitung anbietet.

Funktionsweise Die Properties werden in einer XML-Datei definiert, die mit der in dem Package befindlichen DTD-Datei (properties.dtd) kompatibel ist.
Ist dies geschehen, kann der PropertiesGenerator sowohl die Klasse, als auch eine beispielhafte Propertiesdatei mit Defaultwerten erzeugen.
